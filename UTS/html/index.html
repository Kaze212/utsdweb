Hello Y'all. Nama saya Zidan Wali Arubusman, lahir dan besar di Jakarta Pusat Saat ini, saya sedang menempuh studi S1 di bidang Teknik Informatika, setelah sebelumnya menyelesaikan pendidikan di SMK dengan jurusan Rekayasa Perangkat Lunak (RPL) saya pada dunia coding bermula karena saya merasa coding itu seperti proses menciptakan sesuatu dan membuatnya "hidup". Menulis kode memberi saya kebebasan untuk mewujudkan ide-ide menjadi sesuatu yang nyata dan bermanfaat, yang menurut saya sangat keren.

Saya memiliki beberapa keterampilan teknis di bidang pemrograman yang mendukung minat dan rencana karier saya. Saya terbiasa bekerja dengan HTML dan CSS untuk pengembangan web dasar, serta menggunakan Tailwind CSS untuk mendesain halaman yang lebih modern. Di luar itu, saya juga memiliki pengalaman dalam menggunakan Android Studio untuk mengembangkan aplikasi mobile, dan menguasai SQL sebagai alat untuk pengelolaan basis data. Semua keterampilan ini telah memperkuat pemahaman saya tentang teknologi dan memberi saya pengalaman praktis dalam pengembangan aplikasi.

Secara pribadi, saya dikenal sebagai seseorang yang suka belajar, terutama dalam bidang coding dan matematika. Saya percaya bahwa setiap langkah pertama dalam proses pembelajaran adalah sangat penting dan memberikan fondasi bagi langkah-langkah berikutnya. Saya juga memandang bahwa setiap penolakan atau kegagalan bukanlah akhir dari perjalanan, melainkan bagian dari proses belajar yang membuat kita lebih kuat. Sifat ini membantu saya menghadapi berbagai tantangan, baik dalam pendidikan maupun dalam hidup sehari-hari.

Di masa depan, saya bercita-cita untuk menjadi seorang programmer yang berfokus pada bidang data science. Saya berharap dapat menggunakan keahlian dalam coding dan analisis data untuk menemukan wawasan yang berguna dan berkontribusi pada perkembangan teknologi Dengan tekad dan ambisi ini, saya berusaha untuk terus berkembang, mengasah keterampilan, dan memperluas wawasan agar siap menghadapi tantangan di dunia profesional yang dinamis dan kompetitif.
